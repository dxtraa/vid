Langkah yang bisa kamu coba:

  1. Cek apakah kamu punya API key
    • Biasanya diperoleh dari platform seperti Google AI Studio, OpenAI, atau Hugging Face.
  2. Masukkan API key ke konfigurasi
    • Biasanya ada di file .env, config.js, atau di halaman pengaturan aplikasi.
  3. Pastikan format API key benar
    • Tidak ada spasi tambahan, tidak terpotong, dan masih aktif (belum expired).
  4. Jika pakai layanan berbayar, pastikan kuota/pembayaran masih aktif.
  5. Restart aplikasi setelah memasukkan API key.
